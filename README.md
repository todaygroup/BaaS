# Book-Authoring Agent System (BAAS)

BAAS is a comprehensive system for authoring books using Agentic AI, RAG, and LangGraph.

## Project Structure

Refer to [project-monorepo-scaffold.md](./docs/04_delivery_stack/project-monorepo-scaffold.md) for detailed directory structure.

## Quick Start
1. Install dependencies: `npm install` and `pip install -r requirements.txt` (TBD)
2. Configure environment: Copy `.env.example` to `.env`
3. Run dev server: `npm run dev`
