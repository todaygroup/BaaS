# 챕터 워크스페이스 UI 디자인 (Chapter Workspace UI Design)

## 3.2 챕터 에디터 및 에이전트 패널
- **Timeline**: LangGraph 노드 실행 순서와 현재 위치 표시.
- **Draft Viewer**: 마크다운 렌더링 + 인라인 비평 표시.
- **Agent Input**: 저자의 추가 지시사항(Vibe)을 입력하는 채팅 인터페이스.

## 3.3 인터랙션
- 노드 클릭 시 해당 노드의 입력/출력 로그 및 RAG 출처 확인.
- "Apply Evaluation" 클릭 시 비평 내용을 바탕으로 초안 자동 수정.
